한국전자기술연구원

정보미디어 연구센터는 본 과제의 주관 기관으로 시계열 데이터 처리 패키지화 및 분석 플랫폼 인프라 구성을 담당한다.



daliworks

달리웍스

달리웍스는 클라우드 기반 IoT 플랫폼 전문 업체로 본 과제에서 IoT 데이터의 수집 부터 시계열 데이터 처리 및 분석을 기반한 서비스 제공까지의 시나리오 연구 개발을 담당한다.

wiseitech

위세아이텍

위세아이텍은 인공지능 및 빅데이터 전문 기업으로 본 과제에서 시계열 데이터의 시각화와 구조화, 시계열 특화 AutoML 을 연구 개발한다.

Korea University

고려대학교 산학협력단

Data Science & Business Analytics(DSBA) 연구실은 시계열 데이터 처리의 전반적 전처리 패키지에 대한 연구 개발 및 이상치 탐지에 대한 최신 기술 적용 등에 대해 담당한다.

Kwangwoon University

광운대학교 산학협력단

Neuroengineering and Artificial Intelligence (NeuroAI) 연구실은 시계열 중 생체 신호에 집중하여 데이터 전처리 및 분석을 진행하고 이를 다른 도메인에 활용할 수 있는 기술을 연구 개발한다.

University of Birminghan

영국 버밍엄대학교

버밍엄 대학은 영상 학습 및 분석 전문 연구를 진행하고 있으며, 영상데이터를 시계열화 하여 타 도메인의 데이터와 연합 분석할 수 있도록 처리하는 기술을 연구 개발한다.

Services

GitHub

주요 연구 개발 결과물을 오픈 소프트웨어로 제공하고 있으며, 순차적으로 공개됩니다.

Data Anlytics Server

CLUST Main Server 테스트 가능한 페이지입니다. 현재 개발 테스트 중입니다.

CSV DATA Collector

MetaData Manager

Meta Manager API

CLUST 플랫폼의 메타데이터를 관리할 수 있는 REST API 입니다.
